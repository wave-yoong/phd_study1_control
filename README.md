# phd_study1_control
Control group chatbot for my phd study 1

## Overview
This repository contains the **control group** chatbot implementation for PhD Study 1, which investigates the effect of user control in AI interactions.

### Control Group Characteristics
- **Direct Response**: The chatbot provides immediate answers without intermediate steps
- **No User Control Steps**: Users receive information directly without being asked about information sources, content structure, or additional preferences
- **Same UI Structure**: Shares the same visual interface as the treatment group (phd_study1) for consistency

## How to Use

1. **Open the chatbot**:
   - Simply open `index.html` in a web browser
   - No installation or server setup required

2. **Interact with the chatbot**:
   - Type your question or message in the input box
   - Press Enter or click the send button
   - The chatbot will respond directly with information

## Files

- `index.html` - Main chatbot interface
- `styles.css` - Styling for the chatbot UI
- `script.js` - Chatbot functionality and logic

## Difference from Treatment Group

| Feature | Control Group (this repo) | Treatment Group (phd_study1) |
|---------|--------------------------|------------------------------|
| Response Style | Direct, immediate | Multi-step with user control |
| Information Source Selection | Automatic | User chooses sources |
| Content Structure | Pre-defined | User reviews and modifies |
| User Control | Minimal | High (2-3 intermediate steps) |

## Technical Details

- Pure HTML/CSS/JavaScript implementation
- No external dependencies
- Responsive design for mobile and desktop
- Korean language support

## Study Purpose

This chatbot serves as the control condition to compare against the treatment group where users have more control over the information gathering and presentation process.
